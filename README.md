# 锐科云工具网 - 导航页 & 本地配置编辑器
一个纯前端、无后端依赖的站长工具聚合导航平台，支持可视化本地配置，开箱即用。

---

## 项目结构
- `index.html`    导航展示主页（用户访问入口）
- `edit.html`     配置编辑器（本地管理后台）
- `config.json`   导航数据配置文件

---

## 导航页功能（index.html）
### 1. 工具分类导航
- 按分区展示站长工具，自动统计并显示每个分区工具数量
- 工具以卡片形式展示，包含图标、名称、描述
- 点击卡片直接跳转对应工具页面

### 2. 搜索功能
- 支持按工具名称、描述关键词搜索
- 支持回车搜索 / 按钮搜索
- 无结果时友好提示
- 清空搜索词自动恢复全部分类展示

### 3. 日间 / 夜间模式
- 底部右侧快捷切换
- 模式偏好本地存储，刷新页面自动保留
- 夜间模式自动适配背景、文字、卡片、图标反色

### 4. 动态数据加载
- 从 `config.json` 加载所有站点信息、分类与工具数据
- 自动合并重复分区、过滤空工具分区
- 最多支持 100 个分区展示

### 5. 交互体验优化
- 卡片 hover 放大 + 阴影增强
- 图标加载失败自动显示占位图标
- 底部实时显示分区总数、工具总数统计

---

## 配置编辑器功能（edit.html）
本地可视化管理导航数据，无需服务器，直接生成 `config.json`。

### 1. 配置管理
- 导入配置：上传已有 `config.json`
- 导出配置：在线预览 JSON 内容
- 重置默认：清空数据恢复初始状态
- 密码验证下载：密码 `123456`，生成可直接使用的配置文件

### 2. 站点信息编辑
- 自定义顶部头像 / Logo URL
- 自定义站点名称
- 自定义个性签名（副标题）

### 3. 工具分区管理
- 添加、编辑、删除分区
- 编辑分区支持回车保存、ESC 取消
- 删除分区时工具自动转移到第一个分区
- 自动重名检测，避免重复分区

### 4. 导航工具管理
- 添加工具：分区、名称、图标、链接、描述
- 支持 Font Awesome 图标或图片 URL
- 实时编辑、删除工具项
- 表单校验完善

### 5. 数据统计与限制
- 实时显示分区数量、工具数量
- 分区上限 100 个，导出时自动截断

---

## 使用说明
1. 直接打开 `index.html` 查看导航页
2. 打开 `edit.html` 编辑配置
3. 编辑完成后输入密码 `123456` 下载 `config.json`
4. 将新配置替换到项目目录，刷新导航页生效
